# Map mark app

A microservice application on java 22/spring boot 3

To run: 
- Clone this repository
- cd docker/docker-compose.yml
- docker compose up

For API testing, you can use contents of /postman folder

Used techniligies:
- Java 22
- Spring Boot
- OpenAPI Generator
- MongoDB
- Keycloak
- Docker Compose
